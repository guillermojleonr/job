# Job

Job is a project inteded to be a suite of tools to improve my job seeking tasks, including:

## Features
### CV management
- Generation
- Improvement based on data.
- Formatting
- Transformation into multiple file formats
- Deployment and serving as website.

The main CV will be deployed in [savingl.cl/cv-guillermoleon](https://savingl.cl/cv-guillermoleon)

Alternate CVs can be deployed in other paths.

### Job market analysis
- ETL data from web
- Process data, structure and perform multiple analysis
